## data flow
```
用户消息
    │
    ▼
┌──────────────────────────────────────────┐
│          Channel Handler                 │
│  (飞书/钉钉/微信/Web)                    │
└──────────────────┬───────────────────────┘
                   │
                   ▼
┌──────────────────────────────────────────┐
│           Gateway                        │
│  ├─ 验签/鉴权                            │
│  ├─ 限流                                 │
│  └─ 消息标准化                           │
└──────────────────┬───────────────────────┘
                   │
                   ▼
┌──────────────────────────────────────────┐
│         Agent 路由匹配                   │
│  ├─ 关键字匹配                           │
│  ├─ AI 判断                              │
│  └─ 默认 Agent                           │
└──────────────────┬───────────────────────┘
                   │
                   ▼
┌──────────────────────────────────────────┐
│         Session Manager                  │
│  ├─ 创建/加载会话                       │
│  ├─ 历史消息                             │
│  └─ 会话变量                             │
└──────────────────┬───────────────────────┘
                   │
                   ▼
┌──────────────────────────────────────────┐
│       构建 Request                      │
│  ├─ prompt (Agent 配置)                 │
│  ├─ models (多模型配置)                  │
│  ├─ messages (历史)                      │
│  ├─ context (会话上下文)                 │
│  ├─ knowledge (RAG 召回)                │
│  ├─ skills (技能列表)                    │
│  ├─ mcps (MCP 配置)                      │
│  ├─ a2a (A2A 代理)                       │
│  ├─ tools (工具)                         │
│  ├─ options (超时/重试/沙箱)             │
│  └─ response_format (输出格式)           │
└──────────────────┬───────────────────────┘
                   │
                   ▼
┌──────────────────────────────────────────┐
│        eino-runner                      │
│  ├─ Intent 路由                         │
│  ├─ LLM 调用                            │
│  ├─ Skill 执行                          │
│  ├─ MCP 调用                            │
│  ├─ A2A 调用                            │
│  ├─ Tool 调用                           │
│  ├─ 沙箱执行                            │
│  └─ 响应格式化                          │
└──────────────────┬───────────────────────┘
                   │
                   ▼
┌──────────────────────────────────────────┐
│         响应处理                         │
│  ├─ A2UI 消息                           │
│  ├─ Tool 调用结果                       │
│  └─ A2A 结果                           │
└──────────────────┬───────────────────────┘
                   │
                   ▼
┌──────────────────────────────────────────┐
│         Channel Sender                   │
│  (飞书/钉钉/微信/Web)                   │
└──────────────────┬───────────────────────┘
                   │
                   ▼
              用户收到回复
```
