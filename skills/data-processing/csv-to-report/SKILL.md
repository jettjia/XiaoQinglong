# CSV 到 HTML 报告生成技能

## 触发条件
用户需要分析 CSV 文件并生成可视化 HTML 报告

## 工作流程
1. 使用 parse_file 工具读取 CSV 文件内容
2. 解析 CSV 数据，提取关键信息：
   - 总订单数、总销售额
   - 各类别销售额统计
   - 支付方式分布
   - 订单状态分布
   - 地区销售分布
3. 生成 HTML 报告（使用 Chart.js 绑定图表）
4. 保存 HTML 文件到 /tmp 目录
5. 返回报告文件路径

## 关键指标
- 订单数量统计
- 销售额统计
- 支付方式占比
- 订单状态分布
- 地域分布
