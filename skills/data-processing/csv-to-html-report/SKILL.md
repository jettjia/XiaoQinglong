# CSV 数据分析 HTML 报告生成

## 功能描述
解析 CSV 文件，提取关键统计信息，生成可视化 HTML 报告

## 触发条件
- 用户需要分析 CSV 数据并生成 HTML 报告

## 工作流程
1. 使用 parse_file 读取 CSV 文件
2. 解析数据，计算关键指标：
   - 总订单数、订单总金额
   - 各品类销售额
   - 支付方式分布
   - 订单状态分布
   - 地区销售情况
3. 生成包含统计图表的 HTML 报告文件
4. 输出报告文件路径

## 示例输出
生成 HTML 报告文件，包含数据表格和统计图表

---
# 技能使用指南

当用户说"分析 CSV 文件并生成 HTML 报告"时使用此技能：

1. 首先使用 `parse_file` 工具读取 CSV 文件
2. 使用 Python 或手动解析 CSV 数据
3. 计算统计指标：
   - 总订单数、总销售额
   - 各类别（category）销售占比
   - 支付方式（payment_type）分布
   - 订单状态（order_status）分布
   - 地区（province/city）分布
4. 生成 HTML 报告，包含：
   - 数据汇总卡片
   - 数据表格
   - 统计图表（可用 Chart.js）
5. 保存 HTML 文件到 /tmp 目录
6. 告知用户报告路径
